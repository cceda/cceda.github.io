CMOS Basics & Process Overview

为什么选择 CMOS

  1. 所有 CMOS 的输出都非常一致
  2. 当输入不变时，功耗非常低。只有在状态转换时（0-1；1-0）才会消耗。状态转换的频率越快，则功耗越大。（主要是短路电流和负载的冲放电过程）
  3. 如果采用 CMOS 工艺，模拟IP，RF，和寄存器都可以集成在单个芯片中。
  4. 高阻抗输入，使用电压触发。输入端只会通过很少的电流。
  5. COMS 使用 PMOS 得到完整的VDD， NMOS 得到完整的GND。即输出的电压与 VDD/GND 相同。
  6. 可以在更严酷的温度下工作（-60oC~130C)
  7. 抗噪声能力强。（阈值电压在VDD/2附近）

在深入探讨CMOS基本知识之前，让我们讨论并了解RC的基础。 在大规模集成电路领域里，非常需要完全了解RC，这在复杂的分析中有很大帮助。 我要说的是，如果我们对所有基本概念都非常了解，那么我们可以很容易地分析和解决任何问题。

RC（为什么RC很重要？）

只使用R＆C不能建立完整而准确的CMOS模型。准确地对CMOS建模需要数百个参数（LEVEL 54 BSIM4.0模型）。 但是对于数字设计和PD工程师而言，对MOSFET的抽象层次的了解肯定会有助于理解和分析复杂的电路和问题。 因此，我建议您彻底了解CMOS的RC模型。

电阻效应（MOSFET的宽度）

一个RC电路中，R = 100k-Ohm和C = 10pF。 电流I与电阻R成反比。如果R高，则电流将减小，因此需要更多时间为负载电容器充电。 参见下面的电路，V（n001）[绿色]是电源，V（n002）[蓝色]是电容器两端的电压。 观察电容器两端的电压曲线图，由于电阻较高，因此充电时间较长。

让我们将电阻R降低到10k欧姆。 随着R的减小，I增大，因此电容器C的充电/放电时间减少。 请参考下面的电路。

现在让我们将这种理解扩展到MOSFET。 MOSFET中的漏极电流

Id = u * Cox * W * sq（Vgs-Vt）/ 2L

MOSFET的导通电阻取决于W（宽度）＆L（长度）。 W值越高，MOSFET的电阻和载流能力越低。
“因此，增加MOSFET的宽度可以减少负载电容器的充电和放电时间（充电和放电时间只不过是上升和下降时间而已）”。 在数字电路实现中（综合和PnR），宽度可以通过使用不同的驱动强度单元来改变。 （例如BUF-D1（w = 1u），BUF-D2（w = 2u），BUF-D3（w = 3u）…）”
“通过增加器件的宽度可以增加通过MOSFET的电流，从而减少上升时间/下降时间（负载电容器的充电/充电时间）”

为什么 PMOS 比 NMOS 慢?

大多数答案将停留在“空穴的迁移率小于半导体中电子的迁移率”。 显然，下一个问题将是“为什么？”
为了理解这个概念，让我们回到能带图的基础。


导电电子（自由电子）在导带中传播，价电子（空穴）在价带中传播。 在施加的电场中，价电子不能像自由电子那样自由地运动，因为它们的运动受到限制。 如果其有效质量较小并且散射事件之间的时间较大，则粒子在半导体中的迁移率较大。
因此，电子的迁移率大于空穴的迁移率。 “现在我们了解了为什么PMOS比NMOS慢”
简而言之，与导带中的自由电子相比，“价带中的孔的运动需要更多的能量（电压）”
由于这种迁移率不匹配，因此在设计标准逻辑单元时会出现beta比率。 我们还需要用于时钟网络的特殊单元，其上升和下降时间非常平衡。 为什么时钟单元需要这种平衡？ 我们将在CTS培训课程中对此进行讨论。

制程/工艺概述

大多数半导体巨头在工艺技术方面进行了大量研究。 对超低功耗，高性能，小尺寸和低成本IC的需求不断增长，这使Fab/公司为该研究投入了更多资金。 传统的平面体工艺有许多问题需要进一步扩展（例如漏电流和短通道效应）。 到目前为止，有两种最流行的解决方案。 一种是3D finfet技术，另一种是FD-SOI。 我们将简要讨论这三种工艺技术。

不同类型的工艺技术：

  1. Bulk CMOS
  2. FD-SOI（绝缘子上的全耗尽硅）
  3. FinFET
